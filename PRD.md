# 个人作品集网站 PRD

## 项目目标

构建一个简洁现代的个人作品集网站，用于展示个人品牌、项目经历与联系方式。

***

## 核心功能

### 1. 首页

- 大标题（Name / Title）
- 简介
- 头像
- CTA 按钮（查看项目 / 联系我）

### 2. 关于我

- 详细介绍
- 技能列表

### 3. 项目展示

项目卡片列表，每个卡片包含：

- 项目名称
- 项目截图
- 简短描述
- 技术栈
- 项目链接

### 4. 联系方式

- 邮箱
- GitHub
- 社交媒体链接

***

## 设计要求

- 简洁现代风格
- 深色主题
- 流畅滚动动画
- 移动端适配

***

## 技术栈（暂定）

- Next.js
- Tailwind CSS
- Framer Motion

***

## MVP范围

第一版只完成基础展示网站上线：

- 单页作品集
- 响应式设计
- 部署上线

暂不包含：

- 博客系统
- 后台管理
- 复杂交互
